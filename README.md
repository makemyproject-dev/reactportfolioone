## 🚀 Getting Started

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/makemyproject-dev/reactportfolioone.git](https://github.com/makemyproject-dev/reactportfolioone.git)

2. **Install dependencies:**
   ```bash
   npm install

3. **Run Application:**
   ```bash
   npm run dev

4. **Build:**
   ```bash
   npm run build
